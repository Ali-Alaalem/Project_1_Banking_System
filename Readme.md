# Banking System

This project is a command-line banking system implemented in Java. It allows users to securely manage their accounts, perform transactions, and track their financial history. The system includes multiple user types, such as customers and bankers, and supports different account types and debit cards.

## User Stories

1. **Secure Account Access**
  : As a user, I should be able to interact securely with my bank accounts through a command-line interface.

2. **Transaction History Tracking**
  : As a user, I want to be able to view a history of all transactions to track my spending and deposits.

3. **Account Creation Request**
  :  As a customer, I want to be able to create a new account request so that my account can be approved by a banker.

4. **Banker Approval of Accounts**
  : As a banker, I want to review and approve account creation requests to manage new users efficiently.

5. **Deposit and Withdrawal Limits**
  : As a user, I want my deposits and withdrawals to respect daily limits based on my debit card type to prevent exceeding allowed amounts.

6. **Funds Transfer**
  : As a user, I want to transfer money to my own accounts or to other users' accounts.

7. **Overdraft Protection**
   : As a user, I want the system to handle overdrafts and apply fees or account restrictions automatically to prevent misuse.
